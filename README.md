# 📊 Customer Churn Analysis

## 📌 Project Overview
This project analyzes customer churn data using Python, Pandas, and NumPy.

The analysis helps identify customer behavior, churn patterns, subscription performance, and revenue trends.

---

## 🚀 Features
- Data Cleaning
- Churn Percentage Analysis
- City-wise Churn Analysis
- Subscription-wise Churn Analysis
- Revenue Analysis
- Age Group Analysis
- Data Visualization
- CSV Export

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

---

## 📈 Insights Generated
- Customer Churn Percentage
- High Churn Cities
- Subscription Performance
- Revenue by Subscription
- Age Group Churn Trends
- Top Paying Customers

---

## ▶️ How to Run

1. Install required libraries:
pip install pandas numpy matplotlib

2. Run the project:
python main.py
